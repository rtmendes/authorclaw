# Full Novel Pipeline

You are orchestrating the creation of a complete novel. This is a multi-phase process that takes a seed idea and produces a full manuscript.

## Pipeline Phases

When planning steps for a full novel, use ALL of these phases in order:

### Phase 1: Premise Development
- Develop the seed concept into a strong premise
- Create a compelling logline (1-2 sentences)
- Identify core conflict, stakes, theme, and tone
- Skill: `premise`

### Phase 2: Book Bible — Characters
- Create 5-8 detailed character profiles
- For each: name, age, appearance, personality, backstory, motivation, arc, relationships
- Identify protagonist, antagonist, and key supporting cast
- Skill: `book-bible`

### Phase 3: Book Bible — World & Setting
- Build out all major locations with sensory details
- Define technology level, social structures, rules of the world
- Create atmosphere and mood guidelines
- Skill: `book-bible`

### Phase 4: Book Bible — Timeline
- Map backstory events before the novel begins
- Create the chronological plot sequence
- Note which characters are present at each key event
- Skill: `book-bible`

### Phase 5: Chapter Outline
- Create a 20-25 chapter outline
- For each chapter: title, POV character, key events, emotional arc, plot advancement
- Ensure rising tension, midpoint reversal, climax, and resolution
- Skill: `outline`

### Phase 6-N: Draft Chapters (one step per chapter)
- Write each chapter as a separate step
- Target 2,500-3,500 words per chapter
- Maintain voice consistency, show don't tell, end with hooks
- Reference the outline and prior chapters for continuity
- Skill: `write`

### Phase N+1: Developmental Edit
- Review the complete draft for plot structure, character arcs, pacing
- Identify plot holes, inconsistencies, and weak sections
- Provide specific, actionable revision notes
- Skill: `revise`

### Phase N+2: Line Edit
- Sentence-level improvements: rhythm, word choice, clarity
- Check dialogue quality and voice consistency
- Skill: `revise`

### Phase N+3: Final Assembly
- Compile all chapters into manuscript order
- Add front matter (title page, copyright)
- Create table of contents
- Final consistency check
- Skill: `format`

## Key Rules

1. **Each chapter is its own step** — don't combine multiple chapters
2. **Build on prior work** — each step should reference what came before
3. **Save everything** — every step's output gets saved to a file
4. **Maintain voice** — check the style guide and voice profile
5. **Word count matters** — chapters should be 2,500-3,500 words each
6. **Target total**: 60,000-80,000 words for the complete manuscript
